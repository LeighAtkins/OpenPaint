Test Relative Positioning System

This test verifies that the new relative positioning system maintains geometric relationships during rotation.

Open the browser console to see test results.

What This Test Does:

  1. Creates a horizontal test stroke (line)
  2. Simulates dragging a label to a custom position above the line
  3. Converts the absolute offset to relative positioning (percentage along line + perpendicular distance)
  4. Simulates rotating the stroke by 90 degrees
  5. Recalculates the absolute position from the relative position after rotation
  6. Verifies that the label maintains the correct geometric relationship to the rotated line
