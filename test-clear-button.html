Clear Button Test

Open browser console and visit: https://sofapaint-a5n6vyhn3-leigh-atkins-projects.vercel.app

Test Steps:

  1. Load an image and draw some measurements
  2. Open browser console (F12)
  3. Run: console.log(window.paintApp?.state?.labelBackground)
  4. Click the "Clear" button
  5. Run: console.log(window.paintApp?.state?.labelBackground) again
  6. Check if label backgrounds disappeared

Debugging Commands:

// Check if state exists
console.log('State:', window.paintApp?.state?.labelBackground);

// Check if button exists
console.log('Button:', document.getElementById('labelBackgroundToggleBtn'));

// Test state change manually
if (window.paintApp?.state) {
    window.paintApp.state.labelBackground = 'clear';
    if (window.redrawCanvasWithVisibility) window.redrawCanvasWithVisibility();
}
